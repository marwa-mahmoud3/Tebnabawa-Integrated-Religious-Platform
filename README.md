## 🏛️ System Architecture Overview
The platform is a sophisticated ecosystem consisting of three integrated layers, built to handle dynamic content, real-time streaming, and secure administrative operations:

1.  **User Frontend**: A modern portal developed with **Angular 14**.
2.  **Admin Dashboard**: A comprehensive management panel, also built with **Angular 14**.
3.  **Backend (API)**: A high-performance engine powered by **.NET Core**.

---

## ⚙️ Backend & Security (.NET)
The backend provides a secure and scalable API layer, engineered with professional documentation standards.

### **Key Technical Features:**
* **🔐 Secure Authentication (JWT)**: Implemented **JSON Web Tokens (JWT)** for robust authorization and secure communication between the two Angular applications and the server.
* **📜 API Documentation (Swagger)**: Integrated **Swagger UI** for professional API testing and clear endpoint mapping, ensuring efficient integration.
* **🚀 RESTful API Design**: A clean, structured architecture for high-speed data flow.
* **🛡️ Security Layers**: Role-based access control (RBAC) to protect administrative CRUD operations.

---

## 🛠️ Admin Dashboard (Angular 14 CRUD System)
A dedicated, secure application developed with **Angular 14** to empower the client with full autonomy over the platform.

* **Full CRUD Functionality**: Create, Read, Update, and Delete capabilities for all modules (News, Quran, Brochures, Sheikh's data).
* **🖼️ Media Gallery Management**: Integrated system to manage all visual assets and uploads within a dedicated [Gallery](/Gallery) folder.
* **Live Operations Control**: Real-time management for **Live Video Broadcasting** and **Radio Streaming**.
* **Dynamic Sync**: Updates made in the dashboard are reflected instantly on the user frontend.

---

## 💻 User Frontend Features (Angular 14)
Designed from scratch to provide a premium, fast, and responsive Arabic experience.

### **Core Modules:**
* **Auth System**: Secure Login and Registration modules.
* **Content Hub**: Dynamic News portal, Digital Holy Quran, and Prophetic Brochures (المطويات).
* **Interactive Media**: 
    * 📡 **Live Video Streaming** integration.
    * 📻 **24/7 Prophetic Radio** audio service.
* **Performance**: Custom **Pagination** logic for all heavy data lists to ensure optimal loading speeds.

---

## 🚀 Technical Challenges & Achievements
* **Dual-Angular Architecture**: Developed two separate Angular 14 applications (Client & Admin) sharing a single .NET backend.
* **Custom UI/UX**: Conceptualized and implemented a unique visual identity from scratch.
* **System Independence**: Delivered a 100% dynamic solution, removing the need for technical intervention in content updates.
